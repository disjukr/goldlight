# 2d examples

Examples in this directory focus on the built-in 2D scene model.

- [`basic_rect`](./basic_rect): animated rectangles rendered by the Rust runtime
- [`fills`](./fills): layered filled path composition
- [`gradients`](./gradients): linear, radial, conical, and sweep gradients on filled paths
- [`strokes`](./strokes): stroke join and cap examples
