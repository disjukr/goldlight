# Render Basic Paths

This example exercises the current `@disjukr/goldlight/drawing` implementation against an offscreen WebGPU target and writes `out.png`.

Run from the repository root:

```sh
bun examples/drawing/render_basic_paths/main.ts
```

Run the CanvasKit comparison snapshot:

```sh
bun examples/drawing/render_basic_paths/canvaskit_main.ts
```
