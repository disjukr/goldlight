# `render_drawing_text_modes`

This example renders text through `@disjukr/goldlight/text` and `@disjukr/goldlight/drawing` together and writes `out.png`.

Run from the repository root:

```sh
bun run build:text:native
bun examples/text/render_drawing_text_modes/main.ts
```

Optional verification:

```sh
bun run typecheck
```
